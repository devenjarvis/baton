### Fixed
- Restored repo management: press `R` on the dashboard to switch the active repo, edit per-repo settings (`s`), remove repos (`d` then `d` to confirm), or add new ones (`a`). `o` (open existing branch/PR) is now advertised in the status bar. Note: removing a repo only deregisters it from Refrain's config — the on-disk `.refrain/` directory and any live worktrees are left intact.
